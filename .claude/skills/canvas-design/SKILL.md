# Canvas Design

Create museum or magazine quality visual art through a two-phase process: Design Philosophy Creation, then Canvas Expression.

## Phase 1: Design Philosophy Creation

Users provide subtle input that serves as a foundation without constraining creativity. The process involves creating an aesthetic movement/design philosophy, then expressing it visually with "90% visual design, 10% essential text."

### Generating Visual Philosophy

**Movement naming**: Use 1-2 word titles (e.g., "Brutalist Joy", "Chromatic Silence").

**Philosophy articulation** spans 4-6 paragraphs addressing:
- Space and form
- Color and material
- Scale and rhythm
- Composition and balance
- Visual hierarchy

**Critical guidelines**:
- Avoid redundancy in design aspects
- Emphasize craftsmanship REPEATEDLY — use phrases like "meticulously crafted" and "master-level execution"
- Leave creative interpretation room for visual expression

### Philosophy Examples

- **"Concrete Poetry"**: Monumental form with Brutalist spatial divisions
- **"Chromatic Language"**: Color zones as primary information system
- **"Analog Meditation"**: Texture and negative space with minimal typography
- **"Organic Systems"**: Natural clustering with modular patterns
- **"Geometric Silence"**: Grid-based precision with Swiss formalism influence

### Essential Principles

1. Visual communication takes priority over text
2. Artistic freedom within the philosophy framework
3. Expert craftsmanship presentation
4. Subtle, niche references embedded invisibly — like "a jazz musician quoting another song"
5. Rewards informed viewers while remaining sophisticated for general audiences
6. Each piece tells a cohesive visual story

## Phase 2: Canvas Creation

Creation demands museum or magazine quality work expressing the philosophy:

### Visual Requirements
- Single-page, highly visual outputs
- Repeating patterns and perfect shapes
- Dense mark accumulation suggesting systematic observation
- Sparse, clinical typography
- Limited, intentional color palettes

### Text Handling
- Minimal, visual-first approach
- Context-appropriate scale
- Thin fonts predominantly
- All elements contained within boundaries
- **Nothing falls off the page and nothing overlaps**
- Font sources from `./canvas-fonts` directory when available

### Quality Standards
- Work should appear to require "countless hours" of labor
- Created by "someone at the absolute top of their field"
- "Painstaking attention to every detail"
- Every pixel placement is intentional

## Final Refinement Step

Prioritize polishing existing composition over adding graphics. The approach emphasizes: "How can I make what's already here more of a piece of art?"

## Multi-Page Option

Additional pages should provide creative variations within the design philosophy framework, creating variations resembling "a whole coffee table book" with tasteful narrative progression.
